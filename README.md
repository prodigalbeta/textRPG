# Text RPG
This is my own messed up way of teaching myself game architecture AND C++ at the same time!

## Build
### Requirements
- cmake 3.12+
- the C++ compiler of your choice
### Instructions
1. Clone it wherever the hell you want
2. `cmake -S . -B build`
3. `cmake --build build`
